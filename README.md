# DataExpansion: Mini-framework para Treinamento de IA em Múltiplos Conjuntos de Dados

## Descrição

DataExpansion é um mini-framework Python para treinar modelos de IA em vários conjuntos de dados similares simultaneamente. Este projeto foi criado com foco em aprendizado, mas espero que seja útil para a comunidade. Ele suporta treinamento em múltiplos bancos de dados temporários, o que pode ser útil para testar a robustez do modelo ou simular vários cenários de dados. Atualmente suporta bancos de dados MySQL e SQLite.

## Recursos

* Treinamento de modelos de IA em múltiplos bancos de dados temporários.
* Suporte a MySQL e SQLite para gerenciamento de banco de dados.
* Pré-processamento de dados integrado, incluindo conversão de string e codificação de rótulos.
* Suporte a múltiplas métricas de avaliação (precisão e outras podem ser adicionadas).
* Tratamento básico de erros.
* Logging para rastrear o progresso do treinamento e erros.
* Cálculo de similaridade para vários tipos de dados (numéricos, texto, imagens, listas).
